# v1.0.0 — Known Limitations

- Platform is research-only. No live order execution ships in v1.0.0.
- No broker order routing. Upstox is used read-only for market data.
- CoinDCX integration is public market-data only. No order execution.
- Tokenized metals surfaced via CoinDCX are not physical spot gold or silver.
- Historical backtest results do not guarantee future outcomes.
- Research Lab and Backtest Lab are deterministic research surfaces.
- Some provider datasets may be temporarily unavailable; the dashboard shows a stale/unavailable state instead of fabricated data.
- Billing/license activation requires manual admin verification. There is no automatic payment provider in v1.0.0.
- AI Market Assistant is deterministic template narrative; there is no LLM inference in v1.0.0.
